# [ LunAuth ]
**"The minimalist and transparent authentication mod for NeoForge!"**

---

## 🛡️ Description
**LunAuth** — это легковесный и открытый мод для авторизации, разработанный командой **O.S. TEAM**. Он создан для защиты серверов от кражи никнеймов и обеспечения безопасности игрового процесса в режиме Offline.

Мод ограничивает действия неавторизованных игроков, запрещая перемещение, взаимодействие с миром и выполнение команд до момента входа в систему.

## ✨ Features
*   **Minimalist Design:** Интерфейс в консольном стиле, ничего лишнего.
*   **High Performance:** Оптимизирован для работы на современных процессорах (разработано на AMD Ryzen 5 7600).
*   **Security First:** 
    *   Ограничение по времени на авторизацию (Auth Timeout).
    *   Защита от перебора паролей (Kick после 3-х неудачных попыток).
    *   Полная заморозка игрока и эффект слепоты до входа.
*   **Transparent Storage:** Все данные игроков хранятся в читаемом формате JSON.
*   **Customizable:** Полная настройка сообщений и параметров через `config/LunAuth/settings.toml`.
*   **Clean Structure:** Все файлы мода изолированы в собственной папке внутри конфигов.

## 🚀 Commands
*   `/register <password>` — Регистрация нового аккаунта.
*   `/login <password>` — Авторизация в существующий аккаунт.

## 🛠️ Configuration
Все настройки мода генерируются автоматически при первом запуске в файле:
`config/LunAuth/settings.toml`

Вы можете изменить:
*   Язык сообщений.
*   Тексты всех приветствий и уведомлений.
*   Лимит попыток ввода пароля.
*   Время, отведенное на вход.

## 🏗️ Requirements
*   **NeoForge:** Последняя стабильная версия.
*   **Java:** 21+
*   **Server Side:** Мод является серверным, установка на клиент необязательна.
