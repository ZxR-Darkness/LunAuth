# [ LunAuth ]
**"The minimalist and transparent authentication mod for NeoForge!"**

---

## 🛡️ Description
**LunAuth** — это легковесный и открытый мод для авторизации, разработанный командой **O.S. TEAM**. Он создан для защиты серверов от кражи никнеймов и обеспечения безопасности игрового процесса в режиме Offline.

Мод полностью изолирует неавторизованных игроков, предотвращая любые манипуляции с миром, общение в чате и обеспечивая их полную неуязвимость до момента входа.

## ✨ Features
* **Minimalist Design:** Интерфейс в консольном стиле, ничего лишнего.
* **High Performance:** Оптимизирован для работы на современных процессорах (разработано на AMD Ryzen 5 7600).
* **Enhanced Security (v1.1.0+):** * **Chat Lockdown:** Блокировка отправки сообщений в чат до авторизации.
    * **Invulnerability:** Полная защита игрока от урона (мобы, игроки, окружение) до входа.
    * **Peaceful Interaction:** Неавторизованные игроки не могут наносить урон другим сущностям.
* **Core Protection:** * Ограничение по времени на авторизацию (Auth Timeout).
    * Защита от перебора паролей (Kick после нескольких неудачных попыток).
    * Полная заморозка игрока и эффект слепоты до входа.
* **Transparent Storage:** Все данные игроков хранятся в читаемом формате JSON.
* **Customizable:** Полная настройка всех сообщений и параметров через `config/LunAuth/settings.toml`.

## 🚀 Commands
* `/register <password>` — Регистрация нового аккаунта.
* `/login <password>` — Авторизация в существующий аккаунт.

## 🛠️ Configuration
Все настройки мода генерируются автоматически при первом запуске в файле:
`config/LunAuth/settings.toml`

Вы можете изменить:
* Тексты всех приветствий, уведомлений и системных ошибок.
* Лимит попыток ввода пароля.
* Время, отведенное на вход (таймаут).
* Визуальные эффекты (слепота и др.).

## 🏗️ Requirements
* **NeoForge:** Версия 21.1.229+ (Minecraft 1.21.1).
* **Java:** 21+
* **Server Side:** Мод является серверным, установка на клиент необязательна.
